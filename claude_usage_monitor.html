Setup Instructions

This monitor fetches live data from Claude.ai. To use it:

  1. Open claude.ai and log in to your account
  2. Make sure you're logged in before refreshing this page
  3. The monitor will automatically update every 5 seconds

Note: Due to browser security (CORS), this page must be served from the same domain as Claude.ai, or you need to use a browser extension. Currently showing placeholder data until authentication is successful.
